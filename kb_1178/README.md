# 📚 Base de Conocimiento — Ley Nº 1178

**Ley de Administración y Control Gubernamentales de Bolivia**

> Curso Especializado de la Escuela de Gestión Pública Plurinacional (EGPP)

---

## 📖 Descripción

Esta base de conocimiento recopila y organiza toda la documentación del **Curso Especializado sobre la Ley Nº 1178**, incluyendo temas, subtemas, normas básicas, leyes complementarias y material de referencia. El objetivo es servir como fuente de consulta rápida y estructurada sobre los Sistemas de Administración y Control Gubernamentales del Estado Plurinacional de Bolivia.
